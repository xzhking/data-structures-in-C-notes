# 锐评《数据结构教程》

针对[《数据结构教程：第六版（微课视频 题库版）》](https://baike.baidu.com/item/%E6%95%B0%E6%8D%AE%E7%BB%93%E6%9E%84%E6%95%99%E7%A8%8B%EF%BC%88%E7%AC%AC6%E7%89%88%C2%B7%E5%BE%AE%E8%AF%BE%E8%A7%86%E9%A2%91%C2%B7%E9%A2%98%E5%BA%93%E7%89%88%EF%BC%89/61919761)，进行技术性点评、分析及补充。

对该书并无偏见，“锐评”的原因是讲授该课程需要使用它，而技术/真理越辩越明:)

> [!NOTE]
> 该书混用了C/C++。鉴于本人对C++的偏见，所以讨论均基于C语言。

## 推荐书籍（C语言版）

数据结构基于C语言的相关教材目前发现最好的只有两本：

- [Data Structures and Algorithm Analysis in C, by Mark Allen Weiss, 2nd Edition](https://book.douban.com/subject/4924153/)
- [Algorithms in C, by Robert Sedgewick, 3rd Edition](https://book.douban.com/subject/1869681/)
